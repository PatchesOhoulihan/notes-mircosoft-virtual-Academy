4. Events

A little 101 about the general stuff of jquery

4.1 Attachment and the Event Object

Events give you the opportunity to respond to particular Events in the Browser. JQuery purpose is to make responding to such Event musch easier than plain Javascript. But remember it can only do what Javascript can do, but much more comfortable.

Attaching to Events

  • .on() and .off() New attaching Methods after the big overhaul of version 1.7 so they're the most important!!

    .on() is used to apply an Eventlistener .off() removes it.

  • .bind() and .unbind() legacy Methodes before version 1.7

  • .delegate() and .undelegated() legacy Methodes before version 1.7

  • .one() legacy Methodes before version 1.7

  • .live() and .die() legacy Methodes before version 1.7

  • .trigger() and .triggerHandler() simulating the trigger of a particular event that have this type of Event applied: $('Element-with-event').trigger('event(ex click)');

NICE TO KNOW: You can easily add mulitple Eventhandler to a group of Elements like <div>>. If you than add a new <div>> to the Page it also gets the Eventhandler.

Event Object

Everytime if you attach an Eventhandler to an Element and the Events fires you have the opportunity to get an Event Object. Like this:

            $('#on').on('click', function(event){
                      console.log(event.currentTarget);
                      //Prevent the Event from bubbleing up to the parent
                      event.stopPropagation();
                  });

You can use this Object to get detailed Information where, which and how the Event occours
Helpful properties include:

  • .currentTarget
  • .type
  • .which
  • .timestamp
  • .target
  • .data

4.2 Browser Events

These Event are are important for the Browser himself. Not in particular for the document which is loaded in it. These are basically three stats:

  • .error() Called if an element is not loaded correctly

  • resize When the size of the browser window changes.

  • scroll When the user scrolls to a different place in an element

To use this Methods you have to select the window object, like this: $(window).on('scroll',function(){}). The window Object is the Browser window itself. REMEMBER in contrast the Page you're displaying is the document Object.

4.3 Document Loading Events

  • .load() When an element is being loaded

  • .ready() When the DOM is loaded and ready(minus content like images) Use $(function) as shortcut

  • .unload() Opposide of load, when an element is being unloaded.

4.4 Form Events

  • .change() When an element value changes

  • .focus When an element receives focus

  • .blur() When an element loses focus

  • .focusin() When an element or any nested elements receive focus

  • .focusout When an element loses focus

  • .select() When the user makes a text selection inside

  • .submit() When the user is attempting to submit a form

4.5 Keyboard Events

  • .keydown() when a key is pressed down

  • .keypress()When the browser registers keyboards input (may vary by browser!) Not triggered by shift, esc, delete etc

  • .keyup() When a key returns to the up position.
