3. DOM Manipulation

Shows how to manipulate the DOM shows you some Daily routine Methods

3.1 Element Manipulation

Manipulating Text of an Element, Text and HTML overriding the existing Text in the selected Element.

  • ('p').html('The new Text is awesome') is like innerHTML in plain vanilla js, you can put text but also new Elements

  • ('p').text('The new Text is awesome') is like innerTEXT in plain vanilla js, you can just put text in the Element

  • $('#div p:first').val() Selects the first Paragraph of a div, and returns the Element Value

PERFORMANCE TIP: If you plan to just add some Text use the .text() Methode because it's faster than the .html() Methode.

3.2 Modifying Elements

  • Inside Elements,the Payload is put inside the selected Element

    This 4 Methods work like Sister Methods, you have the choice between the two kinds of Syntax. Append and Prepend Add new Text to the Text of the selected Element

    $('content').appendTo('selector')
    $('content').prependTo('selector')

    $('selector').append('<li>this is after<li>')
    $('selector').prepend('<li>this is before<li>')

  • Outside Elements,the Payload is put in a new Tag before or after the selected Element.

    $('content').insertAfter('selector')
    $('content').insertBefore('selector')

    $('selector').after('<li>this is after<li>')
    $('selector').before('<li>this is after<li>')

    $('selector').before(content, function(index){})

  • Copying Elements

    $('selector').clone().appendTo('selector2'), clone() deep copy an Element append it to another Element.

  • Creating new Elements, alternative to append() or after() and aso Methodes, NOTE: Element created this way is NOT added to the DOM automatically

    
                   $('<a/>',{
                      html: 'This is a <strong>new</strong>',
                      'class': 'link',
                      href:'page.html'
                  });

  • Removing Elements

    $('selector').remove();
    $('selector').detach();
    $('selector')empty();

  • Replacing Elements

    $('selector').replaceAll('content');
    $('selector').replaceWith('content');

  • Wrapping Elements

    $('selector').wrap();
    $('selector').wrapAll();
    $('selector')wrapInner();
    $('selector').unwrap();

Style Elements

  • $('selector').css("style","value");

  • $('selector').css({'style1':'value1','style2':'value2'})

  • $('selector').css("style",function(index,value){});

NOTE: Don't forget this is inlinestyle you're doing with these Methods. It's way better to apply and remove CSS classes and ids to/from the particular Element.

3.3 Modifying Attributes

Adding or Deleting attributes of a selected Element

  • $('selector').attr('name','value');

  • $('selector').prop('name','value');

  • $('selector').removeAttr('name');

  • $('selector').removeProp('prop');

  • $('selector').attr('name', function(){});

NOTE: Attr and Prop in this case mean the same thing, just another Name for the same funcitionality
