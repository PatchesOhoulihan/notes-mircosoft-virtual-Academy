1. jQuery 101

A little 101 about the general stuff of jquery. Jquery is an utility Library that helps you write code with more comfortable than an plan Javascript.

1.1 What is jquery

Makes Javascript easier and the Dom stink less. jQuery is really known to be the write less, do more JavaScript library.

  • Fast, small(32k), open, feature-rich, has alot of constributers and you can be be one too.

  • HTML document traversal and manipulation

  • Event handling

  • Animation

  • Ajax

  • Ver compatible

  • Extensible

  • Cross browser

If you get the Library from a popular cdn there is a high possiblity that the user has already downloaded it.

1.2 When to use jquery?

  • Websites

  • Small to medium-sized web apps

  • Enhancement of brown-field(opposite of green-field application aka Legacycode) web applications

1.3 When not to use?

If you want to build a spa(single page application) for that Angular is the better option.

Don't use it if you only need one or two Features out of it.

2. Selection

Gives some Examples about types of selectors and how to use them.

2.1 Selectors

Selectors are the core Concept of jQuery. Use the $ Dollarsign followed by a CSS Selector example: $('CSS Selector')

Select multiple Elements from the DOM: $('selector1, selector2, selectorN') or $('element[attribute='some'][attribute="other"]')

Select a one Element outof many Elements of the same kind: $('selector')[number of the Element in the DOM Tree] OR $('slecetor').get(Number of the Element in the DOM Tree)

Qjuery has Selectors for all kind of Types:

  • Elements: by id, name etc

  • Attributes

    • $("selector[attr]") Element has this Attribute
    • $("selector[attribute='attrvalue']") Element has this Attribute with this Value
    • $("selector[attribute^'attrvalue']") Element Attribute Value begins with this Value
    • $("selector[attribute$'attrvalue']") Element Attribute Value ends with this Value
    • $("selector[attribute*'attrvalue']") Element Attribute Value contains this Value

  • Form Elements

  • Hierarchy(Parent, Child, Decendant, Sibling)

    • $("ancestor descendant")
    • $("parent elem > child elem")
    • $("prev elem + next elem")
    • $("prev silbling + next sibling")TYPO!!!

  • Elements by state(enabled, disabled, hidden, selected)

  • Position

    • $("element:first") Selects the first Element of a kind
    • $("element:last") Selects the last Element of a kind

  • Class

2.2 JQuery filters

Selectors in JQuery following the CSS Syntax for selecting Things Filters are more like jquerys own thing

$(:button) Selects all buttons
$(":contains(text)") Selects all Elements with the particular Text.
$(':not(selector)')Everything that is not selected this way.

2.3 Chaining

Allows you to run multiple commands on wrapped set while only accessing those DOM elements once, like this $(selector).method1().method2()
Example: $('p').css('background-color','red').css('color','green')

3. DOM Manipulation

Shows how to manipulate the DOM shows you some Daily routine Methods

3.1 Element Manipulation

Manipulating Text of an Element, Text and HTML overriding the existing Text in the selected Element.

  • ('p').html('The new Text is awesome') is like innerHTML in plain vanilla js, you can put text but also new Elements

  • ('p').text('The new Text is awesome') is like innerTEXT in plain vanilla js, you can just put text in the Element

  • $('#div p:first').val() Selects the first Paragraph of a div, and returns the Element Value

PERFORMANCE TIP: If you plan to just add some Text use the .text() Methode because it's faster than the .html() Methode.

3.2 Modifying Elements

  • Inside Elements,the Payload is put inside the selected Element

    This 4 Methods work like Sister Methods, you have the choice between the two kinds of Syntax. Append and Prepend Add new Text to the Text of the selected Element

    $('content').appendTo('selector')
    $('content').prependTo('selector')

    $('selector').append('<li>this is after<li>')
    $('selector').prepend('<li>this is before<li>')

  • Outside Elements,the Payload is put in a new Tag before or after the selected Element.

    $('content').insertAfter('selector')
    $('content').insertBefore('selector')

    $('selector').after('<li>this is after<li>')
    $('selector').before('<li>this is after<li>')

    $('selector').before(content, function(index){})

  • Copying Elements

    $('selector').clone().appendTo('selector2'), clone() deep copy an Element append it to another Element.

  • Creating new Elements, alternative to append() or after() and aso Methodes, NOTE: Element created this way is NOT added to the DOM automatically

    
                   $('<a/>',{
                      html: 'This is a <strong>new</strong>',
                      'class': 'link',
                      href:'page.html'
                  });

  • Removing Elements

    $('selector').remove();
    $('selector').detach();
    $('selector')empty();

  • Replacing Elements

    $('selector').replaceAll('content');
    $('selector').replaceWith('content');

  • Wrapping Elements

    $('selector').wrap();
    $('selector').wrapAll();
    $('selector')wrapInner();
    $('selector').unwrap();

Style Elements

  • $('selector').css("style","value");

  • $('selector').css({'style1':'value1','style2':'value2'})

  • $('selector').css("style",function(index,value){});

NOTE: Don't forget this is inlinestyle you're doing with these Methods. It's way better to apply and remove CSS classes and ids to/from the particular Element.

3.3 Modifying Attributes

Adding or Deleting attributes of a selected Element

  • $('selector').attr('name','value');

  • $('selector').prop('name','value');

  • $('selector').removeAttr('name');

  • $('selector').removeProp('prop');

  • $('selector').attr('name', function(){});

NOTE: Attr and Prop in this case mean the same thing, just another Name for the same funcitionality
