Effects or Animation are Important for the userexperience aslong as there're suttle, or purposeful animation.

Basic Effects

The Basic Effects are:

  • .hide()

  • .show()

  • .toggle() If it's showing hide it, if it's hiding show it.

Fading

Fading adjusts the opacity of the selected element.
The Methods are:

  • .fadeIn()

  • .fadeOut()

  • .fadeTo(), Set the opacity to a set value

  • .fadeToggle(), If it's showing, fade it out. If it's hidden, fade it in.

Sliding

Use the slide Methods to move element around on the Page
The Methods are:

  • .slideUp() Colapse an element using a sliding motion.

  • .slideDown() Shows an element using a sliding motion

  • If it's showing, then hide it. If it's hiding, then show it.

Custom Effects

This Effects are used as a base for all your more complex Effects.
The Methodes are:

  • .animate() Perform a custom animation of a set of CSS properties. You can use all CSS properties that have nummeric value.

  • .queue(), .dequeue(), .clearQueue() Control the queue of functions to be executed on the matched elements

  • .delay() Set a timer to delay execution of subsequent items in the queue.

  • .stop() Set a timer to delay execution of subsequent item in the queue

  • .finish() Stop the animation
