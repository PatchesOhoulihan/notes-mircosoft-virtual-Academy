Whats this Chapter about

What is an Expression

  • JavaScript code snippets placed in bindings {{Hello World}}
  • Evaluated against a scope object
  • Forgiving, don't throw an error
  • Can use filters with expressions to format data
  • Expressions do not have access to window, socument or location
  • Angular Expression Doku

What can you do in an Expression

It is possible to do a onetime bind

Think of a header on a page. That's not going to change. It's always going to be the same header but maybe you need to dynamically populate it for whatever reason. So maybe it's coming from the NAV and the NAV has something associated with it. You got... whatever it may be. You know it's not going to change. You maybe just want to bind that once so that when that whole module runs, gets configured and then runs and that link is made, then it just does it once and it's not worried about watching that item all the time.
